# Asset Placeholder Files

This directory should contain:

- **icon.png** - App icon (1024x1024px)
- **adaptive-icon.png** - Android adaptive icon (1024x1024px)
- **splash.png** - Splash screen image (1242x2436px)
- **favicon.png** - Web favicon (48x48px)
- **click.mp3** - Sound effect for counter increment (optional)
- **goal.mp3** - Sound effect for goal completion (optional)

## Instructions:

1. Replace these placeholder files with actual assets before building the app
2. For sound files, you can use short, subtle sounds (< 1 second)
3. For icons, use simple, clean Islamic geometric patterns or calligraphy

## Recommended Tools:

- **Icons**: Canva, Figma, or Adobe Illustrator
- **Sounds**: Free sound libraries like Freesound.org or create using Audacity
