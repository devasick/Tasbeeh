## 🎨 Color Scheme

### Light Mode
- Background: #F8F9FA (Light Gray)
- Cards: #FFFFFF (White)
- Text: #2C3E50 (Dark Blue-Gray)
- Primary: #2ECC71 (Green)
- Border: #E1E8ED (Light Border)

### Dark Mode
- Background: #1A1A1A (Very Dark)
- Cards: #2C2C2C (Dark Gray)

## 📐 Layout Principles

1. **Large Touch Targets**: All buttons are at least 44x44 points
2. **Clear Typography**: Large, readable fonts (16-72pt)
3. **Generous Spacing**: 16-24pt padding/margins
4. **Visual Hierarchy**: Clear distinction between primary and secondary content
5. **Islamic Aesthetics**: Clean, minimal, geometric design
6. **Accessibility**: High contrast ratios, clear labels

## 🎭 Animations

1. **Counter Increment**: Scale up/down (100ms)
2. **Goal Completion**: Fade in + scale (500ms)
3. **Screen Transitions**: Slide/fade (300ms)
4. **Button Press**: Subtle scale (100ms)
5. **Progress Bar**: Smooth width transition (200ms)

## 📱 Responsive Design

- Works on phones and tablets
- Adapts to portrait and landscape
- Safe area insets respected
- Keyboard-aware layouts
- Flexible grid system

## ✨ Interaction Details

### Haptic Feedback Patterns
- **Light**: Counter increment
- **Medium**: Button press
- **Heavy**: Not used
- **Success**: Goal completion

### Sound Effects
- **Click**: Short, subtle beep (< 0.5s)
- **Goal**: Pleasant chime (< 1s)

### Visual Feedback
- Button opacity change on press
- Progress bar fills smoothly
- Counter scales on increment
- Modal slides in from bottom
- Tab icons change color when active

This visual guide provides a reference for understanding the app's layout and design without actually seeing it running. The actual implementation includes smooth animations, proper touch interactions, and responsive behavior.
